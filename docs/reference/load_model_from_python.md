# Load the neural network model from its .hdf5 file.

Load the neural network model from its .hdf5 file.

## Usage

``` r
load_model_from_python(model_path)
```

## Arguments

- model_path:

  String containing the path to the model's .hdf5 file.

## Value

The loaded model.

## Examples

``` r
if (FALSE) { # \dontrun{
nn_model <- load_model_from_python("/path/to/the/model.hdf5")
} # }
```
